Instrucciones para activar APIs con API key
Claude AI (consulta inteligente):
Para activar respuestas de IA real en la pestaña "Consulta IA", abre este archivo HTML en un editor de texto, busca la línea que dice const CLAUDE_API_KEY = ''; y reemplaza con tu API key de console.anthropic.com. Tu key nunca se envía a terceros — va directo a la API de Anthropic.

Nota de seguridad: Si publicas en GitHub Pages con tu API key en el código, cualquiera puede verla. Úsala solo en uso personal o agrega un backend proxy para producción.

BCRP: No requiere API key. La API es pública y gratuita.
MIDAGRI SISAP: Los datos de precios mayoristas están en el portal público. Esta versión usa datos referenciales actualizables manualmente.
Commodities: Para datos en tiempo real de commodities, registra en commodities-api.com (plan free disponible).
